Create a New File

Now that you've got a repository started let's add a file to it.

Open your text editor and create a new empty file. Now write a little 
bit of text, perhaps type "Hello!", and save the file as 'readme.txt' in 
the 'hello-world' folder you created in the last challenge.

Step: Make More Changes

Now add another line to 'readme.txt' and save the file again.

In terminal, you can view the difference between the file now and how it 
was at your last commit.
